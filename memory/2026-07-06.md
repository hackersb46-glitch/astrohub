4. **CSS 样式缺失**：`.ts-btn` 和 `.ts-btn.active` 无定义，按钮无视觉反馈

**修复方案（v8.50）：**
1. **前端 console.html**：
   - L1682, L1696：`apiPost` → `apiPut`
   - 添加 `.catch()` 错误处理和 toast 提示
   - 添加 CSS 样式（`.triple-switch`, `.ts-btn`, `.ts-btn.active`）
2. **后端 router.py**：
   - `get_filter_settings` / `set_filter_settings` / `get_slow_shutter` / `set_slow_shutter`
   - 同步 `client.get()` / `client.put()` 改为 `await asyncio.to_thread(...)`
3. **constants.py**：VERSION v8.48 → v8.50

**备份：** `backup_20260706_0422/`（console.html, router.py, constants.py）

### 关键发现：dayNightMode HTTP 0 根因
- 设备 XML 中**不存在** `<dayNightMode>` 标签
- 日夜模式由 `<IrcutFilterType>` 控制：`day`/`night`/`auto`/`schedule`
- 设备 capabilities: `<IrcutFilterType opt="day,night,auto,schedule">auto</IrcutFilterType>`
- 官方支持 4 种模式：白天(day)、夜晚(night)、定时(schedule)、自动(auto)
- 老板要求只保留：白天、夜晚、自动
- 设备无独立 dayNightMode 端点（全部 404）
