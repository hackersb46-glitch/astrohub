## 录像功能最终方案（22:03 确定）
- **方案**：FFmpeg 拉取 RTSP 流 → 本地 `astro_hub/record/` 目录
- **原因**：设备无 HDD/SD/NAS，ISAPI ContentMgmt 端点全部不可用
  - PUT `/ContentMgmt/record/control/manual/start/tracks/101` → 200（但需要设备有存储）
  - POST `/ContentMgmt/search` → 403（无存储，无法搜索）
- **FTP 支持**：AstroHub 主动连接 FTP 服务器上传已录制文件（不是设备端直接写 FTP）
- **录音**：PTZ 录像不需要音频，FFmpeg 用 `-an` 去掉（pcm_alaw 不支持 MP4）
- **子进程**：stdout/stderr 必须用 DEVNULL，用 PIPE 会阻塞导致 0 字节
- **RECORD_DIR**：`PROJECT_ROOT / "record"` = `D:\astro_py\astro_hub\record`
- **E2E 测试通过**：5秒录制 → 1.5MB, 4.1s, H.264 3840x2160 MP4
