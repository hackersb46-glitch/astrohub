## Speed测试Tilt速度为0问题修复（20:00-20:52）

### 问题现象
- Speed测试运行完成（108项），但speed.json中Tilt速度全部为0
- `repeats=0`，`movement_direction=unknown`
- 原始CSV有Tilt数据（位置从900→830→...），但只有6个点

### 根本原因
**采样阶段限位检测逻辑错误**：
- 正向测试（direction=1）从上限900开始往下移动
- 原代码检测`direction > 0 and pos >= tilt_max`
- 结果：起点900就触发限位，只采样了几个点就退出
- 数据不足110个点，post_process_csv跳过处理

### 修复内容
**measure_speed_single中的限位检测改为**：
```python
# 正向往下移动，检测是否到达下限
if direction > 0 and pos <= tilt_min:
    hit_limit = True
# 负向往上移动，检测是否到达上限
elif direction < 0 and pos >= tilt_max:
    hit_limit = True
```

### 逻辑说明
- 正向测试（direction>0）：从上限开始往下移动 → 检测下限
- 负向测试（direction<0）：从下限开始往上移动 → 检测上限
