## 关键教训

### 1. 精调方向逻辑要基于反差变化，不基于固定规则
- 旧逻辑：下降2步才反向 → 太慢，导致振荡
- 新逻辑：反差 < best2 立即反向 → 1步检测到越过峰值

### 2. best3与best2的关系
- best2 = 精调起点（回退后的反差）
- best3 = 精调过程中的最高反差
- 要求：best3 >= best2（精调必须比起点更好）

### 3. 粗调平坦区域处理
- 平坦 = 最近5点波动<10%
- 20步平坦 → 第一次反向
- 反向后仍平坦 → 0.5s步长加速通过

### 4. 移动后必须等待0.3s
- 对焦和白平衡都需要
- 确保设备物理稳定后再截图

### 5. 撞墙逻辑
- best1<100 → 直接失败，不进入精调
- 撞墙 = best1≥100 + 过best后下降 + 下降后稳定
- 进入精调的前提是best1≥100
