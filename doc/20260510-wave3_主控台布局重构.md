# Wave 3: 主控台布局重构 + 全局移除设备选择器

**日期**: 2026-05-10
**版本**: wave3-console-layout

## 一、主控台页面布局重构 (page-console)

### 当前问题
- 无左右分区，布局混乱
- 设备选择器占据空间
- 视频流、摇杆、预置点等布局不合理

### 目标布局
```
┌──────────────────────────────────────┐
│         实时画面 (置顶全宽)            │
│  ┌────────────────────────────────┐  │
│  │        video stream            │  │
│  └────────────────────────────────┘  │
├──────────────────┬───────────────────┤
│  指向控制 (左)    │  录像/截图/推流(右) │
│                  │                   │
│  ┌──┐  速度       │  ┌──────────┐    │
│  │↑ │  [====]    │  │  截图     │    │
│  │←→│            │  │  录像     │    │
│  │↓ │            │  │  推流     │    │
│  └──┘            │  └──────────┘    │
│                  │                   │
│  预置点           │                   │
│  [选择][前往]     │                   │
│  [保存]          │                   │
│                  │                   │
│  跟踪控制         │                   │
│  [恒星][月球]     │                   │
│  [跟踪][停止]     │                   │
└──────────────────┴───────────────────┘
```

### 具体修改

#### 1. 移除设备选择器 `#ptzDevice`
#### 2. 视频流区域置顶 (video id=consoleVideo)
#### 3. 左右布局: console-left + console-right
- 左: 摇杆 + 速度滑块 + 预置点 + 跟踪控制
- 右: 截图 + 录像 + 推流

#### 4. CSS 新增
```
.console-layout { display: flex; gap: 16px; }
.console-left { flex: 1; }
.console-right { flex: 1; }
.console-video { width: 100%; margin-bottom: 16px; }
```

## 二、全局移除设备选择器

### 所有页面移除设备选择器
| 页面 | 元素 ID | 操作 |
|------|---------|------|
| 主控台 | `#ptzDevice` | 删除 |
| 高级功能-功能探测 | `#advDetectDevice` | 删除 |
| 高级功能-限位检测 | `#advLimitDevice` | 删除 |
| 高级功能-速度检测 | `#advSpeedDevice` | 删除 |
| 高级功能-推流/存储 | `#advStreamDevice` | 删除 |
| 观测计划 | (如有) | 删除 |

### JS 修改
- 所有 `getElementById('xxxDevice').value` → `connectedDevice.ip`
- connectedDevice 为 null → 提示"请先在设备管理中连接设备"

### 需要修改的 JS 函数
| 函数 | 修改为 |
|------|--------|
| ptzMove/ptzHome/ptzStop/getPtzPosition/toggleOsd | `connectedDevice.ip` |
| funcDetectRun | `connectedDevice.ip` |
| limitTestRun | `connectedDevice.ip` |
| speedTestRun | `connectedDevice.ip` |
| streamStart | `connectedDevice.ip` |
