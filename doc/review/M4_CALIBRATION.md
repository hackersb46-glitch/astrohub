# M4 校准服务

## 概述
设备校准、自动对焦、色彩平衡、速度映射。

## 前置约束
> 版本: v1.0
> 文档用途: 每次运行后逐项自检，达成标准则该项通过，未达成则记录问题
> 开发过程中，前一个P没有通过，不得跳过当前P，必须修复直到通过评审点
> 不允许做任何假设，如有问题，就是评审点不通过
> 任何有明确返回错误代码的，都需要查询错误码具体是什么
> 必须全量搜索已有案例、查询官方文档，解决问题
> 遇到阻滞，第一时间回看官方文档，查找关键词，全量阅读。禁止向人类发问
> 每一个P自检通过评审之后，将正确方法记录在对应method.md表格的对应操作方法中

## 评审明细

| 编号 | 标题 | 内容描述 | 操作方法 | 评审标准 |
|------|------|---------|---------|---------|
| P0 | 校准框架 | 校准流程管理、校准状态机、校准日志 | | |
| P0.1 | 校准流程管理 | 管理校准各阶段的执行顺序 | 定义校准步骤序列，支持前置校验、执行、后校验、回滚 | 步骤按定义顺序执行；前置校验不通过时中止校准；步骤可单独执行 |
| P0.2 | 校准状态机 | 跟踪校准过程的状态流转 | 状态包括:idle→preparing→calibrating→verifying→completed/failed | 状态流转符合定义；每个状态有明确的进入和退出条件；异常时进入failed状态 |
| P0.3 | 校准日志 | 记录校准过程详细信息 | 每步操作记录级别/时间戳/参数/结果，校准结束生成汇总 | 日志包含完整校准过程；可追溯每步操作和结果；汇总包含成功/失败项 |
| P1 | 自动对焦 | 对焦范围探测、对焦精度测试、自动对焦算法 | | |
| P1.1 | 对焦范围探测 | 获取设备对焦能力的上下限 | 获取设备对应端点和操作方式，从最小值逐步测试到最大值，记录有效范围 | 确认对焦端点存在；获得正确上下限（如1~100）；测试后将数值恢复原状；无法获得则评审不通过 |
| P1.2 | 对焦精度测试 | 测试对焦在不同位置的精度 | 在范围内选取5个测试点，分别设置并读取实际对焦位置 | 每个测试点实际值与设定值偏差<5%；记录所有测试点数据 |
| P1.3 | 自动对焦算法 | 实现自动对焦功能 | 通过对比度检测或相位检测算法找到最佳对焦点 | 自动对焦后画面清晰度达到可接受标准；对焦时间<10秒 |
| P1.4 | 对焦还原 | 对焦测试后恢复原始状态 | 记录测试前的对焦值，测试完成后恢复 | 恢复后对焦值与原始值一致（误差<1%） |
| P2 | 色彩平衡 | 白平衡校准、色温调节、色彩还原度测试 | | |
| P2.1 | 白平衡校准 | 校准设备白平衡 | 获取白平衡端点，测试不同色温下的表现，记录最佳参数 | 获得白平衡调节范围；校准后白色物体在画面中呈现为白色 |
| P2.2 | 色温调节 | 调节设备色温参数 | 设置不同色温值（2800K-6500K），验证输出变化 | 色温调节后画面色温相应变化；调节范围覆盖2800K-6500K |
| P2.3 | 色彩还原度测试 | 测试设备色彩还原准确性 | 使用标准色卡，对比设备输出与标准色值 | 色彩偏差在可接受范围内（Delta E<10）；记录各色块偏差数据 |
| P3 | 速度映射 | 电机速度校准、速度曲线拟合、精度验证 | | |
| P3.1 | 电机速度校准 | 校准PTZ电机在不同速度下的实际表现 | 设置不同速度值（1/25/50/100），测量实际移动速度 | 获得速度设定值与实际速度的映射关系；速度线性度可接受 |
| P3.2 | 速度曲线拟合 | 根据校准数据拟合速度曲线 | 使用校准数据拟合设定值→实际速度的函数关系 | 拟合曲线R²>0.9；拟合结果可用于速度补偿 |
| P3.3 | 速度精度验证 | 验证补偿后的速度精度 | 使用补偿后的速度值执行移动，测量实际速度 | 补偿后速度偏差<5%；各速度档位均通过验证 |
| P4 | 位置校准 | 坐标系校准、偏差补偿、精度测试 | | |
| P4.1 | 坐标系校准 | 校准设备P/T坐标系 | 移动到多个已知坐标点，记录实际到达位置 | 获得坐标系偏差；偏差在可接受范围内（P/T偏差<10） |
| P4.2 | 偏差补偿 | 应用偏差补偿 | 根据校准结果建立补偿表或补偿函数 | 补偿后实际到达位置与目标位置偏差<5 |
| P4.3 | 位置精度测试 | 验证补偿后的定位精度 | 移动到10个随机坐标点，记录实际到达位置 | 90%以上坐标点偏差<5；记录所有测试点数据 |
| P5 | 校准数据存储 | 校准参数持久化、校准历史查询 | | |
| P5.1 | 校准参数持久化 | 保存校准结果到持久化存储 | 将校准参数（对焦/色彩/速度/位置）保存为JSON文件 | 保存后可正确读取；文件包含设备MAC/校准时间/各参数值 |
| P5.2 | 校准历史查询 | 查询设备校准历史记录 | 按设备/时间范围查询校准记录 | 返回历史记录列表；每条记录包含完整校准参数 |
| P6 | 校准报告 | 校准结果报告、校准建议生成 | | |
| P6.1 | 校准结果报告 | 生成校准结果报告 | 汇总所有校准步骤的结果，生成MD格式报告 | 报告包含每项校准的通过/失败状态、具体数值、建议 |
| P6.2 | 校准建议生成 | 根据校准结果生成改进建议 | 分析校准数据，识别需要关注的项目 | 建议具体可操作；包含优先级排序 |
| P7 | 校准恢复 | 校准参数恢复、异常回滚 | | |
| P7.1 | 校准参数恢复 | 从保存的校准参数恢复设备 | 读取校准参数文件，逐项写入设备 | 恢复后设备状态与校准时一致；不支持的参数跳过并记录 |
| P7.2 | 异常回滚 | 校准过程中发生异常时回滚 | 校准失败时，将已修改的参数恢复到校准前状态 | 回滚后设备状态与校准前一致；回滚操作本身记录日志 |
