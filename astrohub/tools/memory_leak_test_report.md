### 流程说明
1. Playwright 启动浏览器，导航到 AstroHub
2. 点击"主控台" nav 按钮
3. POST /api/v1/ptz/{id}/record/start 启动录像
4. 等待5秒录像
5. POST /api/v1/ptz/{id}/record/stop 停止录像
6. 点击"回放" nav 按钮，等待5秒
7. 点击"主控台" nav 按钮，回到主控台
8. 实时监控 Python 进程内存
